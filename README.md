# KCity Mini — 시뮬레이터 데모

K-City의 제한된 구역에서 **합성 센서 데이터(synthetic sensor data)** 를 생성할 수 있는 시뮬레이터의 데모 버전입니다.

## 다운로드 및 실행

빌드 파일은 이 저장소를 `clone` 하는 것이 아니라, **[Releases](../../releases)** 페이지에서 내려받습니다.

1. [Releases](../../releases) 탭에서 최신 버전의 분할 파일(`KCity_Mini.zip.001`, `.002`, `.003`)을 **모두** 같은 폴더에 다운로드합니다.
2. `KCity_Mini.zip.001` 을 **7-Zip** 또는 **반디집** 으로 압축 해제합니다. (`.001` 만 풀면 나머지 조각은 자동으로 합쳐집니다. 분할 압축이라 Windows 기본 압축 기능으로는 열리지 않습니다.)
3. 생성된 `KCity_Mini` 폴더에서 `KetiSim.exe` 를 실행합니다.

## 시스템 요구 사항

- Windows 10 / 11 (64-bit)
- 다운로드 약 4 GB, 압축 해제 후 약 11 GB 이상의 저장 공간
